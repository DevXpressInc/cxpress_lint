# avoid_unused_parameters

| Severity | Quick Fix | Options |
|:--------:|:---------:|:-------:|
| Warning  |    ❌     |   ✅    |

## Details

**DO** avoid unused parameters.

Unused parameters should be removed to avoid confusion.

```dart title="Bad"
void log(String a, String b) {
  print(a);
}
```

```dart title="Good"
void log(String a) {
  print(a);
}
```

## Usage

To enable the `avoid_unused_parameters` rule, add `avoid_unused_parameters` under custom_lint > rules in your `analysis_options.yaml` file:

```yaml
custom_lint:
  rules:
    - avoid_unused_parameters
```

## Options

| Option               | Type           | Description                     | Default Value |
| -------------------- | -------------- | ------------------------------- |:-------------:|
| `ignored_parameters` | `List<String>` | A list of parameters to ignore. |     `[]`      |

To configure the list of ignored parameters, add a list of strings to the `ignored_parameters` option:

```yaml
custom_lint:
  rules:
    - avoid_unused_parameters:
      ignored_parameters: ['ref']
```
