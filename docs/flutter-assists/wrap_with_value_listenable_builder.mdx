# wrap_with_value_listenable_builder

Wrap the selected widget with a `ValueListenableBuilder`.

```dart title="Before"
Widget build(BuildContext context) {
  return Text(textNotifier.value);
  //     ^^^^
}
```

```dart title="After"
Widget build(BuildContext context) {
  return ValueListenableBuilder(
    valueListenable: textNotifier,
    builder: (context, value, child) {
      return Text(textNotifier.value);
    },
  );
}
```
