# wrap_with_expanded

Wrap the selected widget with an Expanded.

<Info> This assist is only available when the selected widget is a direct child of a `Flex` widget. </Info>

<Info> This assist is only available when the selected widget is not an `Expanded`, `Flexible` or `Spacer`. </Info>

```dart title="Before"
Widget build(BuildContext context) {
  return const Column(
    children: [
      Text('Hello'),
  //  ^^^^
      Text('World'),
    ],
  );
}
```

```dart title="After"
Widget build(BuildContext context) {
  return const Column(
    children: [
      Expanded(
        child: Text('Hello'),
      ),
      Text('World'),
    ],
  );
}
```
