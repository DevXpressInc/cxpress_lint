# wrap_with_stack

Wrap the selected widget with a Stack.

```dart title="Before"
Widget build(BuildContext context) {
  return const Placeholder();
  //     ^^^^^^^^^^^^^^^^^
}
```

```dart title="After"
Widget build(BuildContext context) {
  return const Stack(
    children: [
      Placeholder(),
    ],
  );
}
```
