# invert_boolean_expression

Invert a boolean expression.

```dart title="Before"
void fn(bool condition) {
  if (number == 0) {
//    ^^^^^^^^^^^
    // ...
  }
}
```

```dart title="After"
void fn(bool condition) {
  if (number != 0) {
    // ...
  }
}
```
